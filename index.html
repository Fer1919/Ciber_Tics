Comandos cmd Linux:

Cd: Cambio de directorio.

Cd .. : Se devuelve un paso en la ruta
Mkdir: Crea directorio.
Touch: Con este comando puedes crear nuevos archivos en el directorio actual.
Mv: mueve un archivo de una carpeta, a otra carpeta.
Ls: muestra el contenido de un directorio.
Este comando lista los archivos y directorios de la carpeta actual.
Para ver archivos ocultos usa ls -a
.soy_un_archivo_oculto mi_archivo.txt mi_carpeta
Para listar el contenido detallado usa -l ls -l -rw-rw-r-- 1 alejandro alejandro 0 oct 22 16:46 mi_archivo.txt drwxrwxr-x 2 alejandro alejandro 4096 oct 22 16:42 mi_carpeta
pwd: Significa Personal Working Directory te muestra la ruta del directorio en que estás ubicado actualmente.
mv (Move) mueve directorios o archivos de una ubicación a otra.
cp (Copy) copia archivos o directorios.
rm y rmdir: El comando rm (Remove) elimina archivos.
cat: te permite leer el contenido de archivos.
find: y la opción -iname puedes encontrar archivos.
clear: limpia la pantalla.
f7: Permite ver el historial de los comandos.

Comandos cmd Windows:

Cd: Cambio de directorio.

BtnWin+regedit: Editor de registro.
Cd .. : Se devuelve un paso en la ruta
Mkdir: Crea directorio.
Type nul>: Crea un archivo.
Xcopy *.* : muve un archivo a una carpeta x.
Dir: muestra el contenido del directorio.
Help: nos mostrará una lista con todos los comandos disponibles.
Systeminfo: muestra información de la configuración del sistema operativo.
Assoc: Este comando muestra o modifica las asociaciones de las extensiones de archivos.
Chkdsk: Comprueba el estado de un disco duro.
Systeminfo: Informa sobre el sistema operativo de un equipo local o remotos.
Assoc: Este comando muestra o modifica las asociaciones de las extensiones de archivos.
Chkdsk. Comprueba el estado de un disco duro.
CMD. Inicia una nueva instancia del intérprete de comandos de Windows.
Dir y Del: Muestra la lista de archivos y subdirectorios de una carpeta.
Driverquery:Este comando lista todos los controladores que tienes en la máquina.
Format. Formatea un disco interno o externo para su uso en Windows.
Powercfg: Informa el estados de ahorro energético, suspensión o hibernación.
Shutdown. Apaga, reinicia o hiberna el equipo local.
Ipconfig: estado de Tcp/Ip.
Tracert: Permite conocer los paquetes que vienen desde un host.
Ping. Prueba el estado de la comunicación del host local.
Pathping: Combina la utilidad de ping y tracert. Es más informativo -+lento-.
Getmac. Obtiene la mac del equipo donde se ejecuta.
Nslookup: Se emplea para conocer si el DNS está resolviendo.
Netstat:Muestra estadísticas de la red y permite diagnósticos y análisis.
Netsh: permite modificar, administrar y diagnosticar la configuración de una red.
